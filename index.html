Hugin is a surveillance dashboard for webmasters. Given a list of domain names, this does a DNS lookup and checks wether the site is marked as "Malicious" by Google's safe browsing API.

About hugin

Hugin is an open-source domain surveillance tool intended as a dashboard for webmasters. You can find the code at github.
